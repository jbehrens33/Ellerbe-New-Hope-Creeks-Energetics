# Ellerbe-New-Hope-Creeks-Energetics
Data and code associated with the manuscript submitted to Limnology and Oceanography titled "Urban Stream Stressors Differentially Constrain Ecosystem Metabolism, Secondary Production, and Insect Emergence"

# Overview
Data and code are provided for estimates of daily metabolism (GPP, ER, K600) in the "Metabolism" folder; secondary production of macroinvertebrates in the "Secondary_Production" folder; and photosynthetically active radiation reaching the stream's surface in the "StreamLight_PAR" folder.

# Usage
Please contact Jonathan Behrens before use, as data is currently in review. Eventually the code and data will be available in Figshare with an associated DOI.
